4. [正方形]
```css
 .wrapper{
        width: 100px;
        background-color: rgb(226, 172, 172);
        border: 1px solid #ccc;
        display: flex;
        /* flex-direction: column; */
        overflow: hidden;
        box-sizing: border-box;
    }
    .wrapper::after{
        content: '';
        padding-top: 100%;
    }
    .wrapper .box{
        display: flex;
        flex-direction: column;
    }
    img{
        width: 70%;
        height: auto;
    }
```
